Como usar o teclado

Vá até o menu principal, localizado na parte superior esquerda de qualquer tela principal e clique em Teclado.
Inicialmente, o teclado tem quadrados com imagens e nomes de categorias.
Ao clicar em um quadrado, você será direcionado para outro teclado mais especifíco com palavras relacionadas à essa categoria.
Este teclado é semelhante ao anterior, tendo quadrados com imagens e palavras
Ao clicar em uma palavra, ela será adicionada à barra com o escrito "FRASE FORMADA", localizada ao topo da tela principal.
Caso você precise de palavras de outras categorias para formar a sua frase, volte para a tela inicial do teclado e escolha o complemento.
Faça isso até formar a frase desejada.

Como criar palavras?

Vá até o menu principal, localizado na parte superior esquerda de qualquer tela principal.
Clique na a opção "Criar Palavra".
A Tela de criação de palavras contém os campos "nome", "imagem" e "categoria", todos devem ser preenchidos.
O campo "imagem" exige o link de uma imagem que represente sua palavra.
O campo "categoria" permite a criação de uma nova categoria. Para mais detalhes, veja _Como criar uma categoria?_ (isso é um link)
Para finalizar, clique no botão "Criar"

Como criar categorias?

Para Criar um Categoria, acesse a página "Criar Categoria" pelo menu lateral, encontrado no canto superior esquerdo.
Ao entrar na tela de criação de palavras, dê um nome para a sua categoria e escolha uma imagem para representá-la.
Você pode escolher sua imagem pela galeria clicando no botão "Acessar a galeria" ou tirar uma foto no momento selecionando "Tirar foto".
Em "Palavras disponíveis", marque as palavras que você deseja que façam parte da categoria criada.
Pressione "Enviar" para criar a categoria ou "Limpar" para desfazer o processo. (Video/imagens/gif)
